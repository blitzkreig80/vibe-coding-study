# 여행 계획 앱

바이브 코딩 스터디를 위해 만드는 여행 계획 앱입니다.

## 📌 프로젝트 소개

여행 일정을 입력하면 자동으로 추천 여행지와 식당을 제시하고, 이동 거리를 계산하여 시간별 스케줄을 작성해주는 편리한 여행 계획 앱입니다.

## ✨ 주요 기능

- **여행 일정 입력**: 여행 시작일과 종료일 설정
- **여행�� 입력**: 방문하고 싶은 여행지 지정
- **추천 여행지**: 입력된 여행지 주변의 추천 여행지 자동 제시
- **추천 식당**: 여행지 주변 식당 추천
- **이동거리 계산**: 여행지 간 이동 거리 자동 계산
- **시간별 스케줄**: 이동시간을 고려한 자동 스케줄 생성

## 🛠️ 기술 스택

- **Frontend**: VS Code (개발 환경)
- **Hosting**: Vercel
- **Language**: JavaScript / TypeScript (예정)

## 🚀 실행 방법

```bash
# 설치
npm install

# 실행
npm start
```

실행파일을 실행하면 바로 앱이 시작됩니다.
